BePresent Εφαρμογή:

Ο κώδικας της εφαρμογής βρίσκεται στο directory bepresent/lib/.
Ο κώδικας της βάσης δεδομένων βρίσκεται στο directory bepresent/SQL/.
Ο κώδικας του API βρίσκεται στο directory bepresent/API/.
Για να τρέξει η εφαρμογή θα πρέπει να κάνουμε source το run.sql (source run.sql;)
στο Directory της βάσης δεδομένων, και να τρέξουμε στο root directory την εντολή
"python -m API.main" για να ξεκινήσει το API.

    bepresent/lib/
        avatar/ #Περιέχει τη σελίδα customisation του Avatar
            avatar.dart
        contacts/ #Περιέχει τη σελίδα customisation των user contacts
            contacts.dart
        game/ #Περιέχει τις σελίδες δημιουργίας session ή
                #σύνδεσης με session, και το παιχνίδι
                #Επίσης περιέχει τις σελίδες για τηλεφωνήματα και μηνύματα
            call.dart
            createsession.dart
            in_game.dart
            joiningsession.dart
            start.dart
            text.dart
        models/ #Όλα τα μοντέλα για την διατήρηση των δεδομένων
                #μέσα στο application καθώς και τις κλήσεις του API
            contacts_database.dart #Διατήρηση δεδομένων contacts μέσα στο app
            inventory_database.dart #Διατήρηση δεδομένων inventory μέσα στο app
            users.dart #Εδώ γίνεται η σύνδεση μέσω API με τη βάση δεδομένων
        stats/ #Περιέχει δεδομένα χρήστη
            stats.dart
        store_pages/ #Περιέχει το menu του store της εφαρμογής και τις σελίδες αγοράς
            clothingpage.dart
            glassespage.dart
            hatspage.dart
            snoutpage.dart
            store.dart
